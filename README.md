# unit-test
Behavior Driven Development Example - Write unit tests for functions with the JavaScript testing framework Mocha.js.

# Installation
```
git clone <repo>
cd unit-test
npm install
```

# Usage
```
npm test
```
or

```
npm run test
```
or

```
mocha 
```
